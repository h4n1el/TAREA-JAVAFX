  Proyecto #1 Registro de Visita
  En este primer proyecto me encargue de agregar los campos necesarios para el requerimiento de la tarea el cual era crear una intefaz para registar visitantes con fecha incluida, tipo de visita y el tipo de documento de estos donde habia una tabla dividida por nombre de campo donde se deberia mostraria la informacion de la visita que se registraria y tambien incluye unos botones para registrar, eliminar y etc

  Proyecto #2 Sistema de Inventario
  Este consiste en desarrollar la interfaz grafica donde pueda registrar productos por su codigo, nombre, categoria y etc donde tambien habia que hace una tabla donde se mostrarian esos productos cuando ya son registrados.Este proyecto tambien cuenta con botones para buscar, eliminar, guadar y editar

  Proyecto #3 Perfil de Estudiante
  Este consistia en crear la interfaz de un perfil de un estudiante y con una figura geometrica simular el lugar donde pone la foto ademas de los campos para poner su matricula, nombre, telefono, direccion y los botones de guardar, imprimir y actualizar
